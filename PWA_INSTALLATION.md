# PWA Installation Guide

## What is a PWA?

Independent Life UK is now a Progressive Web App (PWA), which means you can install it on your device like a native app. It works offline and provides a better user experience.

## Benefits of Installing

- **Quick Access**: Launch directly from your home screen
- **Offline Support**: Access content even without internet
- **Native Feel**: Works like a regular app
- **Auto Updates**: Always get the latest version
- **No App Store**: Install directly from the browser

## How to Install

### On Android (Chrome/Edge)

1. Open https://app-8a0fiym6u1a9.appmedo.com in Chrome or Edge
2. Tap the menu (three dots) in the top-right corner
3. Select "Install app" or "Add to Home screen"
4. Follow the prompts to install
5. The app icon will appear on your home screen

**Alternative Method:**
- Look for the install banner at the bottom of the screen
- Tap "Install" when prompted

### On iOS (Safari)

1. Open https://app-8a0fiym6u1a9.appmedo.com in Safari
2. Tap the Share button (square with arrow pointing up)
3. Scroll down and tap "Add to Home Screen"
4. Name the app and tap "Add"
5. The app icon will appear on your home screen

**Note:** iOS requires Safari browser for PWA installation.

### On Desktop (Chrome/Edge/Brave)

1. Open https://app-8a0fiym6u1a9.appmedo.com in your browser
2. Look for the install icon in the address bar (computer with arrow)
3. Click the install icon
4. Click "Install" in the popup
5. The app will open in its own window

**Alternative Method:**
- Click the menu (three dots) in the top-right
- Select "Install Independent Life UK"
- Follow the prompts

### On Windows

After installing via browser:
- The app appears in your Start Menu
- Pin it to taskbar for quick access
- Works like a native Windows app

### On macOS

After installing via browser:
- The app appears in Applications folder
- Add to Dock for quick access
- Works like a native Mac app

## Troubleshooting

### Install Button Not Showing

**On Android/Desktop:**
- Make sure you're using Chrome, Edge, or Brave
- Clear browser cache and reload
- Check if app is already installed

**On iOS:**
- Must use Safari browser
- Other browsers don't support PWA installation on iOS

### App Not Working Offline

- Visit all pages you want to use offline first
- The app caches content as you browse
- Some features require internet connection

### Update Not Installing

- Close all app windows
- Reopen the app
- Click "Update Now" when prompted
- If issue persists, uninstall and reinstall

### Uninstalling the App

**On Android:**
- Long press the app icon
- Select "Uninstall" or "App info" → "Uninstall"

**On iOS:**
- Long press the app icon
- Tap "Remove App" → "Delete App"

**On Desktop:**
- Click the menu (three dots) in the app window
- Select "Uninstall Independent Life UK"

**Alternative (Chrome):**
- Go to chrome://apps
- Right-click the app
- Select "Remove from Chrome"

## FAQ

**Q: Do I need to install it?**
A: No, the app works fine in the browser. Installing provides a better experience.

**Q: Does it use storage on my device?**
A: Yes, but minimal. Typically 5-20 MB depending on usage.

**Q: Will it work without internet?**
A: Yes, for previously viewed content. Some features require internet.

**Q: How do I update the app?**
A: Updates are automatic. You'll see a notification when an update is ready.

**Q: Can I use it on multiple devices?**
A: Yes! Install on all your devices. Data syncs across devices.

**Q: Is it safe?**
A: Yes, it's as safe as using the website. All connections are encrypted.

**Q: Does it drain battery?**
A: No more than using the website in a browser.

**Q: Can I uninstall it?**
A: Yes, uninstall like any other app. Your data remains in the cloud.
